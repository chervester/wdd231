FC Barcelona Fan Site Plan

Site Name

Barça Central – This name represents a central hub for all FC Barcelona fans, providing news, match details, and interactive content.

Site Purpose

The website serves as a dedicated fan site for FC Barcelona. It provides:

  • Latest team news and updates
  • Player profiles and statistics
  • Match schedules and live stats
  • Fan interactions such as polls, quizzes, and comment sections

Scenarios

Visitors might ask:

  • Who are the top scorers for this season?
  • When is Barcelona’s next match?

Color Schema

Primary Colors:

  • #0033A0 (Barcelona Blue) – Background, main sections
  • #A50044 (Barcelona Red) – Headers, highlights
  • #FFCC00 (Barcelona Yellow) – Accents, call-to-actions

Typography

Fonts:

  • Montserrat – Used for headings
  • Arial – Used for body text

Wireframe

Wireframe sketches will include:

  • Mobile view – A simple navigation menu with quick links to news, matches, and fan zone.
  • Desktop view – A wider layout with a hero banner, featured news section, match schedule, and an interactive fan section.
